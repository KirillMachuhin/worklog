About

This app is a simple calendar for you. You can either use it to keep track of future events or you can use it to record past events. It's free to use and all your data can exported.

You do not need an account. Until you have an account cookies are used to remember your events and if you use different computers (ie. different cookies) you can set up account to save your events to.

You can share your calendar with other people simply by giving them your unique sharing URL.

This web application was built by Peter Bengtsson of Fry-IT, London, England. Project started in September 2010.

The following stack is being used to make things run:

  • Nginx for the front web server
  • Tornado for the application server
  • Python to run the application server
  • MongoDB for the database
  • MongoKit to connection Python to MongoDB
  • jQuery for the client side rendering
